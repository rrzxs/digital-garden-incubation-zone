# 磁力光标 (Magnetic Cursor)

一个交互式的物理引擎演示，光标周围的几何元素会受到磁力吸引或排斥，创造出流畅自然的视觉体验。

## 功能特性

- **物理引擎**：基于速度、加速度和摩擦力的物理模拟
- **磁力效果**：支持吸引和排斥两种模式，可调节强度
- **碰撞检测**：元素之间的弹性碰撞系统
- **拖尾效果**：动态的轨迹追踪，增强视觉连贯性
- **多种形状**：圆形和方形随机分布
- **霓虹美学**：深色背景配合霓虹色彩，营造科技氛围
- **触摸支持**：完美适配移动端设备

## 交互说明

### 模式切换
- **吸引模式**：元素被光标吸引，向光标聚集
- **排斥模式**：元素被光标排斥，逃离光标

### 参数调节
- **磁力强度**：控制磁力的强弱程度（0-100%）
- **元素数量**：调整屏幕上的元素数量（10-100个）

### 操作技巧
- **点击/按住**：增强磁力范围和强度
- **移动光标**：引导元素运动
- **快速移动**：创造更有动力的碰撞效果

## 技术实现

- **纯原生实现**：Vanilla JS + Canvas API，无框架依赖
- **物理模拟**：自定义物理引擎，包含向量计算和碰撞响应
- **性能优化**：使用 requestAnimationFrame 确保 60FPS 流畅体验
- **响应式设计**：自动适配不同屏幕尺寸
